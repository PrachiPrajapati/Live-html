How to manage font category?

To manage (add/edit/delete) font/webfont category,
  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select Web Fonts.
  3. Move to the top-right-corner and tap Manage Categories.
  4. A box will appear. You can take following actions there:
    • Add a category - Under Webfont Categories, simply enter a name and click Add.
    • Edit a category - Hover the mouse over your selected web font category and click on edit icon.
    • Delete a category - Hover the mouse over your selected web font category and click on delete icon.
  5. Once done, tap outside the box to close it.

Your changes have been successfully applied.
