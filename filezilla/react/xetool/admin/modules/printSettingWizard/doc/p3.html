How to apply settings of an existing print profile?

To apply settings of an existing print profile,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Load Print Profile.
  5. A new page will appear. Select your desired print profile from the drop-down.
  6. Click Save or Save & Continue.
You have applied settings of an existing print profile.
