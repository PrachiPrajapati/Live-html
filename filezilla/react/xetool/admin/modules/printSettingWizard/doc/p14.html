How to set price for image/s uploaded by user?

This price is added when your customer add their own image. You can also charge your customer for adding image on each side.

To set image upload price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Price Settings.
  5. Under price settings, find Image Upload Price and enable it.
  6. A pop-up will appear where you need to enter number of sides and click update. Next, set price for each side.
    -- Calculate once per side :
    Price is calculated once irrespective to the number of images a user adds in a single side.
    -- Calculate multiple per side :
    Price is calculated every time a user adds an image in a single side.
  7. Click Save.
Your image upload price will be enabled.

Example:
1. Calculate Once Per Side:
In inkXE Admin Number of Sides updated is 4 and for each side price is set as $1. When image is added on first side then the price will be $1 and in the second side two image is added then again $1 is calculated. So the Image Upload Price is $2.

For First Side one image is added so the price will be $1
For Second Side two image is added so the price will be $1
Then the Image Upload Price will be $2.

So the price structure will be :

Product Price + Image Upload Price = Total price ($20 + $2 = $22)

2. Calculate Multiple Per Side:
In inkXE Admin Number of Sides updated is 4 and for each side price is set as $1. When image is added on first side then the price will be $1 and in the second side two image is added then again $2 is calculated. So the Image Upload Price will be $3.

For First Side one image is added so the price will be $1
For Second Side two image is added so the price will be $2
Then the Image Upload Price will be $3.

So the price structure will be :

Product Price + Image Upload Price = Total price ($20 + $3 = $23)
