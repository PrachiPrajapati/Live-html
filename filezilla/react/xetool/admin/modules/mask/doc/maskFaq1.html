How to add image mask?

To add image mask,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Image Masks.
  3. Move to the right and tap Add Mask.
  4. A box will appear. Upload your mask (svg) and image (png, jpeg, jpg, bmp). Give it a name.
  5. Click Save.
Your image mask has been added.

How to edit image mask?

To edit image mask,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Image Masks.
  3. Move to the right and you will see the available image masks.
  4. Now, select the image(s) you want to edit and click Edit Selected.
  5. A box will appear where you can edit properties of selected image mask, such as name.
  6. Make your changes and click Save.

Your image mask has been edited.

How to delete image mask?

To delete image mask,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Image Masks.
  3. Move to the right and you will see the available image masks.
  4. Now, select the image mask you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete image masks in bulk. Simply select multiple image masks and follow same instructions.

Your image mask has been deleted.
