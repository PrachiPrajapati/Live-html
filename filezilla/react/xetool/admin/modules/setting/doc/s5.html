How to set currency for designer tool ?

To set currency for designer tool,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Price Styling.
  4. Inside, go to Prefix and enter your currency value there. Here, you can also set how a price appears for any product. Simply enter value for Suffix.
  5. Click Save.
You have set currency for designer tool.
