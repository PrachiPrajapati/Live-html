How to create a template?

To create a template,
Save button for template creation will be visible in designer tool only when you are logged in designer admin.
  1. Log into designer admin.
  2. Open inkXE designer tool.
  3. Apply Graphics/text etc on product.
  4. Click on Template Save button.
  5. A pop up window will appear.
  6. Give a name for the Template and assign a category.
  7. Click on Save.
  8. Template Saved Successfully message will appear in designer tool window.
You can view the saved template in template section of graphics in designer admin and template section of designs in designer tool.

How to edit template?

To edit template,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Templates.
  3. Move to the right and you will see the available templates.
  4. Now, select the template you want to edit and click Edit Selected.
  5. A box will appear where you can edit properties of selected template, such as name and category.
  6. Make your changes and click Save.

TIP: You can also edit templates in bulk. Simply select multiple templates and follow same instructions.

Your template has been edited.

How to delete template?

To delete template,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Templates.
  3. Move to the right and you will see the available templates.
  4. Now, select the template you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete templates in bulk. Simply select multiple templates and follow same instructions.

Your template has been deleted.
