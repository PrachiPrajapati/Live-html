How to set number of preloaded items for designer tool ?

To set number of preloaded items for designer tool,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Number of Preloaded Items.
  4. Enter your desired value under each item.
  5. Click Save.
You have set the number of preloaded items for designer tool.
