How to add font/web font?

To add font/web font,

NOTE: Supported format is ttf.

TIP: You can also add web fonts in bulk. Simply select multiple web fonts and follow same instructions.

  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select Web Fonts.
  3. Move to the right and tap Add Web Fonts.
  4. A box will appear. Click Add Webfonts to upload a web font and set its properties, such as name, price, tags, and category.
  5. Click Save.
Your web font has been added.

How to edit font/web font?

To edit font/web font,
  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select Web Fonts.
  3. Move to the right and you will see the available web fonts.
  4. Now, select the web font you want to edit and click Edit Selected.
  5. A box will appear where you can edit properties of selected web font, such as name, price, tags, and category.
  6. Make your changes and click Save.
TIP: You can also edit color palettes in bulk. Simply select multiple web fonts and follow same instructions.

Your web font has been edited.

How to delete font/web font?

To delete font/web font,
  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select Web Fonts.
  3. Move to the right and you will see the available web fonts.
  4. Now, select the web font you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete web fonts in bulk. Simply select multiple web fonts and follow same instructions.

Your font/web font has been deleted.
