How to enable setup price?

INFO: Setup cost allows you to add an optional printing setup amount to the overall printing cost.

To enable setup price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Price Settings.
  5. Under price settings, find Setup Cost and enable it.
  6. Give desired setup price in the box that appears below.
  7. Click Save.
Setup price is enabled now.

There are different condition for setting up the Setup Cost :
  1. If "Per Product Side" is Enable then the price will be added for each side that has been customized. Example :
    The setup price is set as $5 and number of sides customized is 2. Then $5 will get multiplied by 2 and that will give to $10 ($5 x 2= $10). So the Setup Cost is $10 which will be added to the total printing cost.
  2. If "Per Product" is Enable then the price will be added only once independent of number of sides customized. Example :
    The setup price is set as $5 and number of sides customized is 4. Only $5 will get calculated to the total printing cost.
  3. If "Single Order" is Enable then price will be added to each item in order. That means total number of order will get multiplied with the Setup Cost. Example :
    The setup price is set as $5 and there is 50 items in order. Then $5 will get multiplied with 50 and that will results to $250 ($5 x 50 = 250). So the Setup Cost is $250 which will be added to the total printing cost.
  4. If "Multiple order" is Enable then price will be added to the total number of order amount. Example :
    The setup price is set as $5 and there is 50 items in order. Then only $5 will get added as Setup Cost.
