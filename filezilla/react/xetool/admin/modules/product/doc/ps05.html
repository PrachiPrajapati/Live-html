How to set price for variable print area ?


NOTE: Make sure you have enabled Area Price located in the Price Sheet section of Print Profile beforehand.
To set price for variable print area for rectangular shapes,
  1. Open inkXE Admin.
  2. Go to Products in the toolbar towards the left and select Product List from the drop down.
  3. The product list will appear towards the right. Select your product.
  4. In the product page, go to Printing Area.
  5. Click on the dropdown below and select Variable Rectangle Printing Area.
  6. Next, locate Price Per Square Inch and enter your price.
    7. NOTE: Price calculation formula is Default Width x Default Height x price per square inch. In this case, it will be 40x20x0.15 = 120. Make sure you have entered values for default width and height.
  7. Click Save.
Your price has been set.
