How to configure user notes?

To configure user notes,
NOTE: A user note allows your customers to add necessary information related to the product, which you need to pay attention while processing the order.
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Notes.
  4. Enter a description that will show to your customers when they are done customizing.
  5. Set character length on the box beside Number of Characters Allowed. (Your users can add notes only up to the character length you have set.)
  6. Click Save.
You have configured user notes.
