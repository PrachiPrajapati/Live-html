How to enable price per letter?

NOTE: Make sure you have set the price for TextFx beforehand from the Admin Panel.

To enable price per letter,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Price Settings.
  5. Under price settings, find Enable Price Per Letter and enable it.
  6. Click Save.
You have enabled price per letter.

Example:

For the first set of TextFx the price is set at $1.50 and for the second set of TextFxt the price is set at $2.50. When your customer adds three letters from the first set of text FX and two letter from the second set of TextFx, the Letter Price will be $9.50.

For three letters from the first set of TextFx, price is $4.50
For two letters from the second set of TextFx, price is $5.00
Letter Price is $9.50

The price will be shown to the customer is :

Product Price + Letter Price = Total price ($15 + $9.50 = $24.50)
