How to enable multiline text price?

INFO: Text line price is a feature that enables you to set price for each line. Your customers will be billed each time they add a new line.

To enable multiline text line price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Price Settings.
  5. Under price settings, find Enable Text Line Price and enable it.
  6. Click Save.
You have enabled multiline text line price.

Example:

In inkXE Admin Number Of Allowed Lines 3 and for each line price is set as $2. When the user add any text of three lines then $6 will get added and for text of two lines then $4 will get added. So the Text price will be $10.

Text Contains three lines is $6.
Text Contains two lines is $4.
Text Price is $10.

The price will be shown to the customer is :

Product Price + Text Price = Total price ($15 + $10 = $25)
