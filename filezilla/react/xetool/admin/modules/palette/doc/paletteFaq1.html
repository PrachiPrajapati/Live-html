How to bulk upload color palettes?

To bulk upload color palettes,

NOTE: Make sure you have prepared the CSV file list containing all the color names and their hex codes.

  1. Open inkXE Admin.
  2. Go to Color Palettes in the toolbar towards the left.
  3. Move to the right and tap Add Palette.
  4. A box will appear. Click Upload CSV and upload the csv file you have created.
    • To upload RGB, create a csv file with name, color_code, category_name, price (optional).
    • To upload CMYK, create a csv file with name, C-M-Y-K value, hexcode.
    • To upload Pattern, simply bulk select and upload.
  5. Click Save.
Your color palettes have been added in bulk.

How to add color palette?

To add color palette,

NOTE: Color palettes can be added in three forms: RGB, Pattern, & CMYK.

  1. Open inkXE Admin.
  2. Go to Color Palettes in the toolbar towards the left.
  3. Move to the right and tap Add Palette.
  4. A box will appear. Select any one option from RGB, Pattern, or CMYK and choose your color from the color picker. Set properties, such as name, price (optional), and category.
  5. Click Save.
Your color palette has been added.

How to edit color palette?

To edit color palette,
  1. Open inkXE Admin.
  2. Go to Color Palettes in the toolbar towards the left.
  3. Move to the right and you will see the available color palettes.
  4. Now, select the color palette you want to edit and click Edit Selected.
  5. A box will appear where you can edit properties of selected color palette, such as name, price, category, and CMYK values.
  6. Make your changes and click Save.
TIP: You can also edit color palettes in bulk. Simply select the color palettes and follow same instructions.

Your color palette has been edited.

How to delete color palette?

To delete color palette,
  1. Open inkXE Admin.
  2. Go to Color Palettes in the toolbar towards the left.
  3. Move to the right and you will see the available color palettes.
  4. Now, select the color palette you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete color palettes in bulk. Simply select multiple color palettes and follow same instructions.

Your color palette has been deleted.
