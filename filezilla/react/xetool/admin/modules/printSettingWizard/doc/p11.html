How to enable font price?

The Font Price is the price for adding premium fonts. If enabled the price will be added to the printing cost. When your customer uses that specific font to customize the product, then price for specific font will be added to the printing cost. The price is added each time when you add the font on the product.

NOTE: Make sure you have set the price of all your premium fonts.

To enable font price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Price Settings.
  5. Under price settings, find Enable Font Price and enable it.
  6. Click Save.
Your font will be priced from now.

Example:
For a specific Font, price is set as $1.50. When your customer use the same Font while customizing the product, then price set for the font will be added to the total printing cost.
So the price will be showing to the customer is :

Product Price + Font Price = Total Price ($15 + $1.50 = $16.50)
