How to enable/disable features of designer tool?

To enable/disable features of designer tool for a print profile,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In a new page. Click FEATURES ON/OFF from left tabs.
  5. You will see a new page towards right with all available features. Uncheck features which you don't want to assign to print profile.
  6. Click Save & Continue.
Features have been enabled/disabled to the print profile successfully.
