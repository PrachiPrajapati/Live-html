How to set maximum image upload size?

To set maximum image upload size,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap File/Image.
  4. Inside, there are 3 textboxes. Here:
    • Enter a value for maximum file size
    • Enter a value for maximum image width
    • Enter a value for maximum image height
    Note: Your users will not be able to upload files that are larger than the values specified by you.
  5. Click Save.
You have set maximum image upload size.
