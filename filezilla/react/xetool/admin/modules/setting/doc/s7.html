How to set add to cart terms and conditions ?

To set add to cart terms and conditions,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move to the right, find Add to Cart Terms and Conditions On/Off and click on it.
  4. Inside, there are two options. Choose Enable and enter a description under Terms and Conditions.
  5. Click Save.
You have set add to cart terms and conditions.
