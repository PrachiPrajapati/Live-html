How to enable used colors?

To enable Used Colors,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Color Settings.
  5. Under color settings, find Show Used Color and enable it.
  6. Scroll down and click Save or Save & Continue.
Used colors will be shown from now.

How to set maximum number of colors or patterns?

To set maximum number of colors or patterns,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Color Settings.
  5. Under color settings, find Maximum Number of Colors/Patterns and enable it.
  6. A popup will appear. Enter your desired value for maximum number of colors/patterns.
  7. Scroll down and click Save or Save & Continue.
You have set maximum number of colors or patterns.

How to set default fill color of text?

To set default fill color of text,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to Color Settings.
  5. Under color settings,scroll down and go to Default Fill Color.
  6. Tap on the color picker and select your desired color.
  7. Click Save or Save & Continue.
You have set default fill color of text.
