How to set minimum order quantity?

To set minimum order quantity,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to General Settings.
  5. Under general settings, find Minimum Order Quantity and enable it.
  6. Enter your desired minimum value inside the textbox.
  7. Click Save & Continue.
You have set minimum order quantity.

How to enable auto conversion of uploaded images?

To enable auto conversion of uploaded images,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, go to General Settings.
  5. Under general settings, find Auto Convert Uploaded Images and enable it.
  6. Select an option from the drop-down.
  7. Click Save or Save & Continue.
You have enabled auto conversion of uploaded images.
