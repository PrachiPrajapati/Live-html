How to set currency for inkXE admin ?

To set currency for inkXE admin,
NOTE: Do not mistake it as currency for designer tool. The currency you set here will get reflected in the designer admin only.
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Currency.
  4. Inside, there is a textbox. Enter currency of your choice.
  5. Click Save.
You have set currency for inkXE admin.
