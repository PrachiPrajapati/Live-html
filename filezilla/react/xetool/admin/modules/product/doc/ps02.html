Pre-decorated Product Template

How to restrict editing options of a template ?

To restrict editing options of a template,
  1. Open inkXE Admin.
  2. Go to Products in the toolbar towards the left and select Product List from the drop down.
  3. The product list will appear towards the right. Select your product.
  4. Click Create Pre-Decorated Product, located at top-right-corner in the product page.
  5. A pop screen will appear of the designer tool. Create your template here.
  6. Once done, select a design, text, uploaded image or anything for which you'd like to restrict editing.
    7. For example, we added a clipart text below. And click Save Pre-Decorated product.
  7. After saving popup bow of Lock properties will appear with locking feature of each design element.
    8. Each item has its own set of distinct properties to lock.
    • For clipart/design, admin can restrict color edit, delete, move, resize, and rotate.
    • For text, admin can restrict color edit, delete, move, resize, rotate, font change and text edit.
    • For uploaded image, admin can restrict delete, move, resize, rotate, mask, filter, reduce color, and select color.
  8. Give a Template name and click Save..
NOTE: To add it is a product, go to inkXE Admin >> Products >> Pre-Decorated products.
Your restrict editing options of a template has been saved.
