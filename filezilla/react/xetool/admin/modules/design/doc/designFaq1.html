How to add clipart/artwork/design?

To add clipart/artwork/design,

NOTE: Supported formats are svg, png, jpeg, jpg, and bmp.

TIP: You can also add cliparts in bulk. Simply, select multiple clipart designs and upload.

  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Designs.
  3. Move to the right and tap Add New Clipart.
  4. A box will appear. Click Upload Design. Set properties of the design you upload, such as name, height, width, category, etc.
  5. Click Save.
Your clipart/artwork/design has been added.

How to edit clipart?

To edit cliparts/designs,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Designs.
  3. Move to the right and you see the available cliparts.
  4. Now, select the clipart you want to edit and click Edit Selected.
  5. A box will appear where you can edit properties of selected clipart, such as name, price, height, width, tags, category, and subcategory.
  6. Make your changes and click Save.
TIP: You can also edit cliparts in bulk. Simply select multiple cliparts and follow same instructions.

Your clipart has been edited.

How to delete clipart?

To delete cliparts/designs,
  1. Open inkXE Admin.
  2. Go to Graphics in the toolbar towards the left and select Designs.
  3. Move to the right and you see the available cliparts.
  4. Now, select the clipart you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete cliparts in bulk. Simply select multiple cliparts and follow same instructions.

Your clipart has been deleted.

How to import clipart?

Note: You will receive a link from inkXE to download the .zip file clipart package.

  1. Open inkXE Admin.
  2. Go to the toolbar section to the left, then click Graphics.
  3. From the drop-down menu select Designs.
  4. In the Graphics page, click Import Clipart.
  5. Select the .zip file provided by inkXE from your local system and click Open.
  6. Click Upload.
Note: You have to set ‘max file size’ to 100MB in your server before importing the clipart package. However, the categories and subcategories are already assigned to the clipart.
