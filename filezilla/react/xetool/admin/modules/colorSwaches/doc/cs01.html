Color swatches

How to add new color swatches ?

  1. Open inkXE Admin Panel.
  2. Go to the toolbar section to the left, then click Products.
  3. From the drop-down menu, select Color Swatches.
  4. Go to the top left corner and, click Add New Color.
  5. Give a color name. Pick a Color from the color picker option or add an Image in .png, .jpg, and .jpeg format.
  6. Click Save.
NOTE: These color swatches are for the products which will reflect in the Designer Tool.
You have added a new color variant for your product.

How to edit color swatches ?

  1. Open inkXE Admin Panel.
  2. Go to the toolbar section to the left, then click Products.
  3. From the drop-down menu, select Color Swatches.
  4. Pick a color from the list you want to change and, click Edit.
  5. Make your desired changes either by picking a Color from the color chart or an Image in .png, .jpg, .jpeg format.
  6. Click Save.
You have edited a color variant for your product.
