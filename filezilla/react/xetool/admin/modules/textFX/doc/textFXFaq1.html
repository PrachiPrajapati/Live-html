How to add text effect?

To add text effect,

NOTE: Supported format is svg.

TIP: You can also add text effects in bulk. Simply select multiple text effect designs and upload.

  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select TextFx.
  3. Move to the right and tap Add New TextFx.
  4. A box will appear. Click Upload TextFX Fonts to upload a text effect and set its property, such as name.
  5. Click Save.
Your text effect has been added.

How to edit text effect?

To edit text effect,
  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select TextFx.
  3. Move to the right and you will see the available text effects.
  4. Now, select the text effect you want to edit and click Edit Selected.
  5. A new page will appear where you can edit properties of selected textfx, such as:
    • Change name
    • Change word
    • Delete textfx
  6. Make your changes and click Save.
Your text effect has been edited.

How to delete text effect?

To delete text effect,
  1. Open inkXE Admin.
  2. Go to Text in the toolbar towards the left and select TextFx.
  3. Move to the right and you will see the available text effects.
  4. Now, select the text effect/s you want to delete and click Delete Selected.
  5. A pop-up will open asking you to confirm your action.
  6. Click Confirm.
TIP: You can also delete text effects in bulk. Simply select multiple textfx and follow same instructions.

Your text effect has been deleted.
