How to set number of elements allowed on print area?

To set the maximum number of elements allowed on print area,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Number of Elements.
  4. Enter your desired value under each feature.
  5. Click Save.
Note: 'Total' represents the maximum number of designs you can add under each element.
You have set the number of elements allowed on print area.
