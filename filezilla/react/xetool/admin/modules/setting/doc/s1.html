How to set language of inkXE designer tool?

To set/change language of inkXE designer,
  1. Open inkXE Admin.
  2. Go to Settings in the toolbar towards the left.
  3. Move right to locate and tap Language.
  4. Inside, there is a drop-down. Click on it and select the language of your choice.
  5. Click Save.
You have set a new language for inkXE designer tool/admin.
