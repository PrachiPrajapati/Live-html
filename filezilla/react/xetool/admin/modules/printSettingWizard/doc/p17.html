How to enable team price?

To enable team price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, select Price Sheet.
  5. Under price settings, click Side Price tab.
  6. Under Side Price, find Enable Team Price and enable it.
  7. After enabling team price, assign print quantity range and team price each for name and number.
You have enabled team price.
Note: Similarly you can add multiple print quantity range and team price for each range by clicking on the + Add Range button

Example:
Hypothetical Question 1:
For an order placed for 39 tshirts priced at $11 each, using name in front and number in back, what should be the Total Price of the product?
Answer: Here,
Order Quantity = 39
Name Price = $0.42
Number Price = $0.52

So, Name & Number Price = Name Price + Number Price
= $0.42 + $0.52
= $0.94

The price will be shown to the customer is :

Total Price = Product Price + Name & Number price
= ($11 * 39) + $0.94
= $429 + $0.94
= $429.94

Hypothetical Question 2
For an order placed for 150 tshirts priced at $16 each, using name and number in front as well as in back, what should be the Total Price of the product?
Answer: Here,
Order Quantity = 150
Name is used twice so name price = 0.82 + 0.82 =1.64
Number is used twice so number price = 0.92 + 0.92 = 1.84

So, Name & Number Price = Name Price + Number Price
= $1.64 + $1.84
= $3.48
The price will be shown to the customer is :

Total Price = Product Price + Name & Number price
= ($16 * 150) + $3.48
= $2400+ $3.48
= $2403.48

How to enable side price?

To enable side price,
  1. Open inkXE Admin.
  2. Go to Print Profile in the toolbar towards the left.
  3. Select your print profile and click on the edit icon.
  4. In the new page, select Price Sheet.
  5. Under Side Price, find Enable Side Price and enable it.
  6. Enter the required number of sides and click Update. Set print quantity range and price per side.
You have enabled side price.
Note: Similarly you can add multiple print quantity range and team price for each range by clicking on the + Add Range button

Example:
Hypothetical Question 1:
For an order placed for 5 t-shirts priced at $16 each and number of sides customized is 2, what should be the Total Price of the product?
Answer: Here,
Order Quantity = 5
Number of Sides = 2

So, side price = Side 1 + Side 2
= $0.26 + $0.58
= $0.84

The price will be shown to the customer is :

Total price = Product Price + Extra Side Price
= ( $16 * 5) + $0.84
= $80 + $0.84
= $80.84

Hypothetical Question 2

For an order placed for 50 tshirts priced at $15 each and number of sides customized is 4, what should be the Total Price of the product?
Answer: Here,
Order Quantity = 50
Number of sides = 4

So, Extra Side Price = Side 1 + Side 2 + Side 3 + Side 4
= 0.69 + 0.55 + 0.20 + 0.10
= 1.54

The price will be shown to the customer is :

Total price = Product Price + Extra Side Price
= ($15 * 50) + $1.54
= $750 + $1.54
= $751.54
